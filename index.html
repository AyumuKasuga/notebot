jabber бот для быстрого сохранения заметок, в данный момен для хранения используется sqlite. Для работы необходимы модули xmppy и sqlite3. В дальнейшем планируется возможность добавления события с временем в котороое нужно о них НЯпомнить. Можно потестровать здесь: notebot@xmpp.ru

dump db

CREATE TABLE "notes" (
    "id" INTEGER PRIMARY KEY AUTOINCREMENT,
    "date" INTEGER,
    "jid" TEXT,
    "text" TEXT
)
CREATE INDEX "jid" on notes (jid ASC)

Доступные команды

  • !help - эта справка
  • !ls - список 15 последних заметок
  • !ls all - список всех заметок (на самом деле только 500 последних)
  • !v 25 - вывести полностью текст заметки 25
  • !r 25 - удалить заметку 25
  • !remove all! - удалить вообще все заметки (осторожно, дополнительно ничего не спросит)
